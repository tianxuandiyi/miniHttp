# miniHttp
本软件主要实现了简单的http服务器相关的代码主要包含两个方面，

1.主程序模块：建立TCP类型的socket在80 端口进行监听请求，接受到了监听请求之后，将链接请求传送给处理模块，并继续进行监听：
